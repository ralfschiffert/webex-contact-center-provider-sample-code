# Media Forking Documentation

This directory contains the documentation for Webex Contact Center Media Forking, published via GitHub Pages.

## View the Documentation

The documentation is available at:
https://ralfschiffert.github.io/webex-contact-center-provider-sample-code/

## Local Development

The documentation is written in Markdown and uses Jekyll for GitHub Pages.

To view locally:
```bash
# Install Jekyll (if not already installed)
gem install bundler jekyll

# Serve locally
cd docs
jekyll serve
```

Then open http://localhost:4000 in your browser.

## Files

- `index.md` - Main documentation (Getting Started Guide)
- `_config.yml` - Jekyll configuration for GitHub Pages theme
